# EKODRIX - Premium Software Studio Website

A world-class, conversion-focused website positioning EKODRIX as a premium software studio competing with global leaders like Vercel, Linear, and Stripe.

## 🚀 Features

- **Premium Design System**: Sophisticated minimalism with purposeful motion
- **Advanced Animations**: Framer Motion, GSAP, and smooth scroll effects
- **SEO Optimized**: Complete meta tags, structured data, sitemap, and robots.txt
- **Performance First**: Optimized for Lighthouse scores > 95
- **Responsive**: Mobile-first design that works on all devices
- **Blog System**: Complete blog infrastructure with SEO optimization
- **Case Studies**: Showcase portfolio with detailed results
- **ResellerPro Showcase**: Flagship product presentation with pricing

## 🛠️ Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript (Strict Mode)
- **Styling**: Tailwind CSS + CSS Variables
- **Animations**: Framer Motion + GSAP + Lenis
- **3D Graphics**: Three.js / React Three Fiber (ready for integration)
- **Deployment**: Vercel (optimized)

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## 📄 Pages

- `/` - Homepage with hero, services, case studies, testimonials
- `/about` - About page with company mission and values
- `/services` - Detailed services overview
- `/work` - Case studies and portfolio
- `/resellerpro` - Flagship SaaS product showcase
- `/blog` - Blog listing page
- `/blog/[slug]` - Individual blog posts
- `/contact` - Contact form
